# WASM-gameboy-camera

- https://germangb.github.io/wasm-gameboy-camera/

Web-based Game Boy camera (Nintendo please don't sue me)

Built with https://github.com/germangb/rust-gameboy

## Controls

| Button | Keyboard Key |
|:---   |:---|
| `A`    | `Z` |
| `B`    | `X` |
| `Start`| `Enter` |
| `Select` | `Right Shift` |
| `D-Pad` | `Arrow Keys` |

## Features

- Everything works except the exposure control
